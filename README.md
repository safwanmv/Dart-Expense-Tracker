# Expense Tracker (Dart)

A simple Dart application for tracking expenses.

## How to Run

Make sure you have Dart installed. Then run the following command in your terminal:

```bash
dart expense.dart
````

## Requirements

* Dart SDK (>=2.12.0)

## Setup

1. Clone the repository.
2. Navigate to the project directory.
3. Run the Dart file using the command above.
